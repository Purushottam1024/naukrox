## [1.0.0] — 2026-04-05

### Added
- Initial open-source release 🎉
- Agentic pipeline: Query Builder → Job Fetcher → JD Analyzer → Resume Tailor → Cover Letter → Tracker
- Flask web app with dark UI
- MongoDB integration for jobs, resumes, and application tracking
- Selenium-based LinkedIn scraper with mock fallback
- GPT-4o powered JD analysis and resume tailoring
- Docker Compose setup with 3 services (web, mongo, selenium)
- MIT License
- CONTRIBUTING.md, CODE_OF_CONDUCT.md, SECURITY.md
- GitHub Actions CI (lint + Docker build check)
- Issue templates and PR template
